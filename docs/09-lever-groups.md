# Imperium Aeternum — 杆组（Lever Group）设计隐喻

> 核心思想：玩家看到的不是一个孤立按钮，而是一组联动的杠杆——拉一根，另外几根必然动。
> 这是"国家扩张越大治理越难"的数学载体。

## 一、最小杆组：税率杆组

```
玩家拉杆：税率 ↑
    │
    ├─→► 国库金 +        （短期收益）
    │
    ├─→► 民众满意度 −    （杆组联动 1）
    │      └─→► 不满度 +
    │            └─→► 叛乱风险 +
    │
    ├─→► 商人满意度 −    （杆组联动 2）
    │      └─→► 贸易收入 −  （长期反噬短期）
    │
    └─→► 人口增长 −      （杆组联动 3，跨回合）
           └─→► 下一回合税基 −  （自反回路）
```

关键性质：税率杆拉得越高，回弹力越大（腐败吞收入、民众反弹）。这是杆组的阻尼。

## 二、全游戏五大杆组

| 杆组 | 主杆 | 联动杆 | 阻尼来源 | 设计目的 |
|------|------|--------|---------|---------|
| 财政杆组 | 税率/建设 | 满意度/腐败/贸易 | 腐败吞收入 | 防穷兵黩武 |
| 军事杆组 | 征兵/宣战 | 厌战/粮耗/人口/稳定 | 厌战→叛乱 | 防无限扩张 |
| 扩张杆组 | 占省/附庸 | 行政超载/文化冲突/合法性 | overExtension罚 | 防涂色 |
| 改革杆组 | 政策/法律 | 派系反弹/改革冲击 | 派系满意度 | 防一键改 |
| 外交杆组 | 结盟/制裁 | 信任/威胁值/贸易依赖 | 被包围网 | 防独狼 |

## 三、三种耦合结构

### 图 A — 串型杆组（链式反噬）

```
   ┌─────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
   │ 宣战  │───►│ 厌战 +  │───►│ 稳定 -   │───►│ 叛乱 +  │
   └─────────┘    └──────────┘    └──────────┘    └──────────┘
                        │
                        └────────► 粮耗 + ──► 人口 -
```
> 一根拉到底，末端必然炸。玩家要学会中途松手。

### 图 B — 环型杆组（自反馈回路）

```
        税率 ↑
       ╱        ╲
      ▼          ▼
   金 +        民众满意度 −
      │          │
      │          ▼
      │        人口增长 −
      │          │
      └◄─────────┘
        税基 ↓  ← 下一回合金 -（反噬）
```
> 环型 = 杆组有"记忆"。今天的爽=明天的痛。这是 Imperium 区别于涂色游戏的核心。

### 图 C — 对冲杆组（双杆博弈）

```
   ┌── 拉杆甲：安抚民众 ──► 满意度 + │ 成本：金 - │ 副作用：腐败 +
   │
   └── 拉杆乙：镇压叛乱 ──► 叛乱 -   │ 成本：粮 - │ 副作用：军方 + / 民众 -

        甲、乙不能同时拉满 → 玩家做权衡
```
> 没有免费午餐。两根杆方向相反，玩家选节奏而非选"对错"。

## 四、杆组在代码里的体现

每个杆组 = 一个 engine 文件 + 一组 formulas 函数。
联动通过 settleXxx 互调实现：

```
turn.ts  processTurn(state)
   ├── settleEconomy          ← 财政杆组
   ├── settlePopulation       ← 人口杆组（读经济结果）
   ├── settlePolitics         ← 政治杆组（读不满度）
   ├── settleCultureReligion  ← 文化杆组（读稳定度）
   ├── settleWars             ← 军事杆组（读厌战/粮）
   └── buildReport            ← 把所有杆的位移汇总给玩家看
```

每个杆组都有一项"阻尼系数"随省份数上升：
- overExtensionPenalty（扩张杆组）
- 腐败吞收入（财政杆组）
- 厌战累积（军事杆组）
- 文化冲突（扩张杆组）

杆不会断，但会越来越重。这正是设计圣经"国家扩张越大治理越困难"的代码化。

## 五、玩家心智模型

玩家不需要看公式。玩家感受到的是：

1. 拉某根杆很爽（金多了、省多了）
2. 但下一回合杆变重了（腐败涨了、不满涨了、厌战涨了）
3. 必须松一根、拉另一根，做节奏切换
4. 长期把任何一根顶死 = 崩盘

这就是 Imperium 的核心乐趣：不是赢，是撑住。
