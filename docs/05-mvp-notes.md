## 2. 关键修复记录

### DEC-019 大国经济/稳定度死亡螺旋（W4）
- 问题：S 级大国 50 回合后稳定度=0、粮食=-39434
- 修复：粮食消耗率 pop×0.5→×0.1；超管惩罚限幅 5/回合；稳定度<30 时 +5 回归力；maxProvinces 按 tier

### DEC-020 AI B/C 级轻量决策激活（W4）
- 问题：B/C 级 101 国完全被动
- 修复：Lite 档加建农田/市场/贸易决策

### v2-A3 A 级稳定度调参 v3（2026-06-24）
- 问题：A 级稳定度长期 ~46，卡在 55 平衡点
- 修复：分段回归力 4/2.5/1 → 5/3/1.5，阈值 55/70 → 60/75
- 效果：A 级稳定 46→**74.3**

### v2-A4/A5 AI 缺粮建农场（2026-06-24）
- 问题：9 国负粮，D 级极穷国永远翻身无望
- 修复：A4 门槛 food<0 → food<consume×2；A5 Static 档加被动建农场（免金）
- 效果：负粮国 9→**1**

### v2-遗留 validate 科技级数 + 重复事件 id（2026-06-24）
- 问题：validate 硬检查科技 5 级但已扩到 8；3 事件 id 重复
- 修复：validate 接受 5 或 8 级；3 重复 id 改名

### v2-A1 PLAYER_ID 全量替换（2026-06-24）
- 问题：PLAYER_ID='n01' 硬编码散布 40 处，选非 n01 剧本即崩
- 修复：13 screen + store + ai + init 全改 state.playerNationId；保留 data/nations.ts 常量作 fallback
- 效果：多剧本选国解锁

## 3. 新机制落地记录

### C1 事件链（2026-06-24）
- 激活 `applyEffect` 的 `triggerEvent` 分支（原本是死字段）
- 加 3 链 9 事件：瘟疫链（爆发→蔓延→余波）、王位继承链（储君薨→夺位→新储）、边境冲突链（边民→对峙→议和）
- 链首 weight>0 可自然触发，链中/尾 weight=0 只靠 triggerEvent
- 2 个新测试：triggerEvent 入队 + 数据完整性

### C2 法律树（2026-06-24）
- 新建 `data/laws.ts`：12 法律 3 类（民法/刑法/行政法）
- 互斥机制：土地改革↔贵族特权、中央集权↔地方自治、严刑↔宽刑
- `engine/politics.ts` 加 `enactLaw` + `lawPerTurnEffects`（每回合不满/叛乱削减）
- PoliticsScreen 加政策/法律页签切换

### C3 贸易路线（2026-06-24）
- 新建 `data/trade-routes.ts`：8 路线（丝绸之路/地中海商路/琥珀之路等）
- 长度修正：短途×1.0 / 中途×1.3 / 长途×1.6
- 失效条件：两端省份都不归玩家
- `engine/economy.ts` 加 `establishTradeRoute` + `availableTradeRoutes` + `routeYieldEstimate`
- EconomyScreen 加路线显示 + 建立按钮

### C4 更多剧本（2026-06-24）
- `generateWorld(seed, playerId, regionFilter)` 加区域筛选参数
- 新增 3 区域剧本：W3 东亚风云（3 洲 ~50 国）、W5 地中海争霸（4 洲 ~70 国）、W6 新大陆崛起（美洲 ~20 国）

### v2-A4 贸易门槛放宽（2026-06-24）
- S/A 档贸易门槛 relation>5 → relation>-10（容忍轻度敌对商贸，符合历史）
- 预期：贸易条约数从 284 进一步提升
