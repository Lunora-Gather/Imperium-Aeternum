## 二、可玩性方案（分优先级落地）

### P0：战争演出系统（WarNarration）
**问题**：宣战→进度条→占领，无故事感。战争是帝国兴亡核心，必须有色。
**方案**：
1. 战争结算时生成 **战报条目**（ArmyBattleReport）：回合、双方兵力、士气、地形修正、胜负、伤亡、转折点
2. MilitaryScreen 战争卡加 **"战报"按钮**，弹窗显示该战所有战报
3. 关键转折记入 **史册**（首胜/惨败/占领敌都）
4. 战报用叙事文案（"第3年，我军于平原伏击敌援，斩敌200，士气大振"）

**改动范围**：military.ts 加战报生成 + MilitaryScreen 加战报弹窗 + chronicle 加战事里程碑
**风险**：低，纯增量不破坏现有结算

### P1：外交关系网络图（DiplomacyGraph）✅ E14 已落地
**问题**：192 国关系是一张大表，玩家难感知"谁是盟友/敌人/贸易伙伴"。
**方案**：
1. DiplomacyScreen 加 **"列表/图谱"切换按钮**：玩家居中，其他国家按关系值环形布局
2. 边色：同盟金/贸易绿/战争红虚线/中立灰点线
3. 仅显示与玩家有显式关系且 |relation|>5 的国家，按 |relation| 排序取前 24
4. SVG 实现，节点 hover 看 title 详情

**改动**：DiplomacyScreen.tsx 加 `DiplomacyGraph` SVG 组件 + view state 切换
**验证**：typecheck ✅ + 48/48 tests ✅

### P1：科技树可视化（TechTree）✅ E14 已落地
**问题**：当前列表，无分支依赖感，科技选择缺路径感。
**方案**：
1. TechnologyScreen 三分支改为 **纵向树状图**：每列一条分支主线（渐变色）
2. 已学节点金边 + 研发中脉冲光晕（box-shadow + ia-pulse 动画）+ 锁定灰
3. 左侧分支节点圆点 + 节点间垂直连线，路径感清晰

**改动**：TechnologyScreen.tsx 重构布局加分支主线 + 节点圆点 + 连线
**验证**：typecheck ✅ + 48/48 tests ✅

### P2：危机紧张感（CrisisEscalation）✅ E14 已落地
**问题**：稳定度从 50→20→崩溃是突变，玩家无渐进预警。
**方案**：
1. Dashboard 警报横幅加 **危机等级**：warn（稳定度<30 或 1 danger）黄边、critical（稳定度<20 或 ≥3 danger）红边脉冲 + "☠ 社稷危急"标识
2. 全屏 inset boxShadow 紧迫反馈（critical 用 ia-crisis-pulse 1.8s 脉冲）
3. 连续赤字分级提示（2年黄/3年红 Tag）
4. 史册加"乱世的序章"（稳定度跌破20）+"财政危机"（连续3年赤字）里程碑

**改动**：Dashboard.tsx 加 crisisLevel + deficitStreak + 全屏危机层 + chronicle.ts 加危机里程碑 + index.css 加 ia-crisis-pulse 动画
**验证**：typecheck ✅ + 48/48 tests ✅

### P2：事件链反馈（ChainHint）✅ E13 已落地
事件弹窗底部加 **"✦ 此事后续或有变…"** 金色斜体标记（选项含 triggerEvent 时）
