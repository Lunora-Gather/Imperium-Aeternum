## §2 经济系统公式

### §2.1 税收

```
税收 = 人口 × 基础税率 × 税收效率 × 稳定度修正 × 腐败修正 × 同化度修正
```

| 变量 | 范围 | 来源 |
|------|------|------|
| 人口 | 0-∞ | Province.population（总和） |
| 基础税率 | 0-0.5 | Nation.taxRate |
| 税收效率 | 0.5-1.5 | Nation.efficiency + 行政科技修正 |
| 稳定度修正 | 0.3-1.2 | stability/100 映射（<30 急剧降） |
| 腐败修正 | 0.5-1.0 | 1 - corruption/200 |
| 同化度修正 | 0.7-1.0 | 0.7 + assimilation/300 |

稳定度修正细分：stability≥60→1.2；40-60→1.0；30-40→0.8；<30→0.5

### §2.2 粮食产量

```
粮食产量 = 农业基础值 × 农民数量 × 地形修正 × 农业科技修正 × 基础设施修正
```

| 变量 | 范围 | 来源 |
|------|------|------|
| 农业基础值 | 0.5-2.0 | Province.agriBase（per capita） |
| 农民数量 | 0-∞ | Province.peasantCount |
| 地形修正 | 0.3-1.5 | terrain：plain 1.2、hill 0.8、mountain 0.3、coast 1.0、forest 0.6、desert 0.3 |
| 农业科技修正 | 1.0-1.4 | 1 + agriLv×0.08 |
| 基础设施修正 | 1.0-1.5 | 1 + farmBuildings×0.1 |

### §2.3 腐败吞税

```
腐败吞税 = 税收 × corruption / 200
实际入库 = 税收 - 腐败吞税
```

corruption 0-100；corruption=100 时吞 50%。

### §2.4 贸易收入

```
贸易收入 = 基础贸易值 × 市场数修正 × 贸易协定数修正 × 商业性格修正 × 贸易依赖安全修正
```

| 变量 | 范围 | 来源 |
|------|------|------|
| 基础贸易值 | 50-200 | Nation 起始 |
| 市场数修正 | 1.0-2.0 | 1 + marketCount×0.1 |
| 贸易协定数修正 | 1.0-2.0 | 1 + tradeDealCount×0.15 |
| 商业性格修正 | 1.0 或 1.25 | commerce≥70 → 1.25 |
| 贸易依赖安全修正 | 0.5-1.0 | 任一贸易国 relation<0 时按依赖度扣 |

### §2.5 建筑产出

每建筑定义 baseYield（{gold?, food?, wood?, iron?, sciPt?, adminPt?, influence?}）+ level。

```
实际产出 = baseYield × level × 省份修正 × 国家修正
```

省份修正：terrain 适配（如矿山在 mountain →×1.5，在 plain →×0.3）。

## §3 政治系统公式

### §3.1 合法性变化

```
Δlegitimacy = +政体巩固 +2
            + 派系平均满意度修正 (avgSat-50)/25
            - 腐败惩罚 (corruption-30)/10  // corruption>30 才扣
            - 厌战惩罚 warExhaustion/20
            - 改革冲击（改革当回合 -10，后续 3 回合每年 -2）
            ± 事件效果
clamp 0-100
```

### §3.2 稳定度

```
Δstability = +合法性修正 (legitimacy-50)/20
           + 派系加权满意度修正 (Σfaction.power×faction.sat - 2500)/100
           - 全国平均不满度 / 5
           - 厌战度 / 10
           - 叛乱省份数 × 3
           ± 事件效果
clamp 0-100
```

派系加权：Σ(power × satisfaction)，理想 = Σ(power)×50，约 2500。

### §3.3 改革成功率

```
改革成功率 = 0.3 + legitimacy/200 + (行政能力)/100 × 0.2 + 主导派系满意度/200
```

失败后果：合法性 -15，主导派系满意度 -10。

### §3.4 可管理省份数

```
可管省 = 3 + adminLv + 行政能力/25 + 中央集权修正(if centralization≥70: +2)
```

超出可管省 → 每超 1 省，全国税收效率 -5%、稳定 -2。

## §4 人口系统公式

### §4.1 阶层人口分布（初始）
国家起始给定每省各阶层比例。发展阶段不再调比例，只调总数。

### §4.2 阶层征兵转化

```
征兵 N 人 → 农民 -N×0.7、工人 -N×0.2、商人 -N×0.1 → 士兵 +N
```

### §4.3 人口增长

```
Δpop = pop × 基础增长率 × 粮食充足度 × 稳定度修正 × 战争修正 × 疾病修正 × 福利修正
```

| 变量 | 范围 | 默认 |
|------|------|------|
| 基础增长率 | 0.005-0.02 | 0.01（1%/年） |
| 粮食充足度 | 0.3-1.5 | food/(pop×0.8)，cap 1.5 |
| 稳定度修正 | 0.3-1.2 | stability/100 映射 |
| 战争修正 | 0.5-1.0 | 战争中 0.7 |
| 疾病修正 | 0.5-1.0 | 瘟疫事件触发时 0.5 |
| 福利修正 | 1.0/1.1 | welfare≥70 → 1.1 |

### §4.4 阶层满意度变化

```
Δsatisfaction = -税率压力 (taxRate-0.15)×100
              - 征兵压力 (本回合征兵数/总pop)×200
              - 粮食短缺 (food<pop×0.5 时 -15)
              ± 阶层特定事件
              + 安抚行动 (+8)
              + 政策匹配 (+5/-5)
              + 国家性格修正
```

每阶层有 policyMatch 字段（政策是否合其意）。

### §4.5 阶层叛乱风险

```
阶层叛乱风险 = (100 - satisfaction) × power/100 × 国家稳定度倒数修正
```

稳定度<20 时 ×2，稳定度>60 时 ×0.5。

## §5 军事系统公式

### §5.1 战斗力

```
战斗力 = 兵力 × 士气/100 × 训练度/100 × 装备/100 × 科技修正 × 将领修正 × 补给修正 × 地形修正
```

| 变量 | 范围 | 来源 |
|------|------|------|
| 兵力 | 0-∞ | Army.size |
| 士气 | 0-100 | Army.morale |
| 训练度 | 0-100 | Army.training |
| 装备 | 0-100 | Army.equipment |
| 科技修正 | 1.0-1.4 | 1 + milLv×0.08 |
| 将领修正 | 0.7-1.3 | general/50（general 35-65） |
| 补给修正 | 0.5-1.0 | supply/100 |
| 地形修正 | 0.7-1.3 | 防守方主场加成 |

### §5.2 叛乱风险（省份级）

```
叛乱风险 = 不满度 × 文化差异修正 × 宗教差异修正 × 驻军压制修正 × 稳定度修正
```

| 变量 | 范围 | 来源 |
|------|------|------|
| 不满度 | 0-100 | Province.unrest |
| 文化差异修正 | 1.0/1.5 | dominantCulture ≠ stateCulture → 1.5 |
| 宗教差异修正 | 1.0/1.3/2.0 | tolerant 1.0 / state 1.3 / persecute 2.0（异教时） |
| 驻军压制修正 | 0.3-1.0 | 1 -驻军数/1000，floor 0.3 |
| 稳定度修正 | 0.5-1.5 | (100-stability)/50 |

### §5.3 战斗结算（每回合推进战争进度）

```
双方战斗力比 = attackerPower / (attackerPower + defenderPower)
Δprogress = (双方战斗力比 - 0.5) × 40  // 进攻方优势
attacker 兵力损失 = defenderPower/attackerPower × 基础损失(5%) × 兵力
defender 兵力损失 = attackerPower/defenderPower × 基础损失(5%) × 兵力
双方士气 -5，厌战 +3
```

progress ≥ 100 → 进攻方占领省份；progress ≤ 0 → 进攻方撤退。

### §5.4 战争代价（每回合）

```
金消耗 = 兵力 × 0.5
粮消耗 = 兵力 × 1.0
厌战 +2
稳定 -1
兵力自然损耗 = 兵力 × 2% （伤病死）
```

## §6 外交系统公式

### §6.1 关系自然漂移

```
Δrelation = +贸易协定(+2) + 同盟(+1) - 边境冲突(-3 if adjacent) - 威胁感(-threat/20) ± 共同敌人(+3 if common enemy)
clamp -100~100
```

### §6.2 威胁值

```
threat(他对玩家) = 玩家军力 / max(他军力, 1) × 30 + 玩家省份数 / 10 × 20 - 关系/2
clamp 0-100
```

玩家军力越强、省份越多、关系越差 → 他越感威胁。

## §7 科技系统公式

### §7.1 科研点产出

```
sciPt = 学者数量 × 0.5 + 学院建筑数 × 2 + 科技性格修正(if technocracy≥70: ×1.25)
```

### §7.2 研发成本

```
costSci = baseCost × level²  // level 1:100, 2:400, 3:900, 4:1600, 5:2500
costGold = baseCost × level × 50
```

## §8 文化宗教系统公式

### §8.1 同化度变化

```
Δassimilation = +基础设施(每道路+1) + 文化政策(assimilate:+2 / respect:0)
              + 行政能力/50
              - 文化差异(-1 if dominantCulture ≠ stateCulture)
              - 远离首都距离惩罚(-1 if 距首都>2省)
clamp 0-100
```

### §8.2 忠诚度变化

```
Δloyalty = +稳定度修正(stability-50)/30
         + 同化度修正(assimilation-50)/30
         + 驻军(+5 if 驻军>0)
         - 不满度/10
         + 福利政策(+3 if welfare≥70)
clamp 0-100
```

### §8.3 文化/宗教差异修正

见 §5.2 叛乱公式中的修正项。

## §9 国家性格累积公式

```
Δtendency_X = 行为权重表[action][X]
自然漂移：每 10 回合所有 tendency -2（向 0 漂移）
激活：tendency ≥ 70 → 激活特性
```

激活后直接修改对应公式的修正系数（如 militarism 激活 → §5.1 士气修正 ×1.10、征兵 N ×1.20）。

## §10 胜负判定公式

### §10.1 胜利（每回合末检查，任一达成即胜）

1. 征服：ownedProvinces ≥ 7 且 stability ≥ 40
2. 经济：gold ≥ 2000 且年贸易收入 ≥ 500，连续 10 回合
3. 文化：influence ≥ 150 且 ≥ 3 国 relation ≥ 70
4. 永恒：连续 200 回合无失败

### §10.2 失败（任一达成即败）

1. 国库破产：gold ≤ 0 连续 3 回合
2. 全国崩溃：avgStability ≤ 10 连续 3 回合
3. 首都被占：capital.ownerId ≠ self
4. 革命成功：任省 rebellionRisk ≥ 100 且驻军=0
5. 国家分裂：≥3 省同时叛乱独立
6. 合法性归零：legitimacy ≤ 0

---

## §11 行动点公式

```
每回合行动点 = 3 + 行政能力修正(floor(adminPt/30)) + 改革政策加成
每行动消耗 1-2 行动点（见 02-system-rules 各系统）
```
