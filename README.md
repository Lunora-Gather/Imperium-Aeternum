## 简介

**Imperium Aeternum** 是一款大策略 / 国家治理模拟游戏。

你不是单纯的将军，也不是只负责扩张的君主。你需要在财政、粮食、人口、法律、派系、战争、外交、科技、地方治理和长期稳定之间做取舍。

游戏的核心不是“征服一切”，而是回答一个更困难的问题：

> 当国家越来越大、系统越来越复杂时，你能否让它继续运转下去？

---

## 核心体验

### 目标教练 → 总参判断 → 风险检查 → 执行 → 年报复盘

游戏现在形成了一套面向公开预览的策略闭环：

```text
目标教练
  → 帝国总参
  → 推进前风险中枢
  → 经济 / 外交 / 战争顾问
  → 玩家执行行动
  → 年度结算
  → 年报复盘
  → 下一年修正计划
```

每一年都不是简单地按下“下一回合”。

你会先看到：

- 下一步最应该做什么
- 当前是否有阻断下一回合的硬风险
- 财政、粮食、补给、稳定和腐败是否可承受
- 外交上应贸易、结盟、修复关系还是防御
- 是否存在安全的战争机会
- 是否应该先存档再推进

## 特色系统

| 系统 | 说明 |
|---|---|
| **目标教练** | 把当前局势转成可执行任务，告诉玩家先点哪里、做什么、为什么 |
| **帝国总参** | 汇总行动建议，给出当前最重要的治理方向 |
| **推进前风险中枢** | 在结束本年前检查待决事件、破产、粮荒、骚动、外交和战争风险 |
| **经济内政顾问** | 判断财政、粮食、补给、稳定、腐败和地方秩序 |
| **外交顾问** | 判断外交态势、影响力、贸易、同盟、战争和威胁对象 |
| **战争机会顾问** | 扫描合法相邻目标，提供宣战前胜率、备战度和风险判断 |
| **战争预演** | 宣战前显示预计胜率、后勤、财政、厌战、外交风险与存档建议 |
| **年度复盘** | 年报后总结本年做对了什么、哪里恶化、下一年该修正什么 |
| **存档体检** | 读档前检查存档健康度，支持恢复与风险提示 |
| **测试与 Pages 门禁** | 使用 TypeScript、Vitest 和 Pages 构建保证公开预览稳定 |

## 四条胜利路线

| 路线 | 目标 |
|---|---|
| **征服路线** | 扩张疆土，同时维持足够安定 |
| **富国路线** | 建立长期财政优势，并连续多年保持稳定 |
| **合纵路线** | 积累影响力，经营友邦与外交网络 |
| **永恒路线** | 在长期和平、安定与法统中延续国家机器 |

真正困难的地方在于：

> 每条胜利路线都会被另一套系统牵制。  
> 征服会带来治理压力，富国会受到腐败拖累，外交会被战争打断，永恒则要求长期克制。

## 项目结构

```text
src/
  components/   # 通用 UI 组件与策略顾问面板
  data/         # 国家、省份、事件、建筑、政策、法律等数据
  engine/       # 回合结算、战争、外交、AI、事件、王朝等核心逻辑
  gameplay/     # 目标教练、风险中枢、顾问、路线图、复盘、行动建议
  screens/      # 总览、地图、年报、存档、外交、军事等页面
  store/        # 全局状态、存档、迁移和游戏入口
  types/        # TypeScript 类型定义
```

## 当前版本重点

当前版本聚焦于公开预览前的完整策略引导体验：

```text
打开游戏
  → 看目标教练
  → 看帝国总参
  → 看推进前风险
  → 处理经济 / 外交 / 战争建议
  → 存档
  → 推进下一年
  → 年报复盘
```

目标不是只做一个“能运行的 demo”，而是让玩家每一回合都能理解：

- 我现在哪里危险？
- 我下一步应该点哪里？
- 我为什么不能贸然扩张？
- 我什么时候可以安全推进？
- 我该如何把国家机器维持下去？
