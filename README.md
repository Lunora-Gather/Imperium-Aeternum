# Imperium Aeternum · 永恒帝国

> 治理一个国家数百年。扩张越快，崩溃越早。真正的胜利不是征服世界，而是建立一个能长期运转的国家机器。

**Imperium Aeternum**（拉丁文「永恒帝国」）是一款以国家治理为核心的策略模拟游戏。玩家不是单纯的将军，而是一个国家的统治者——管理政治、经济、人口、军事、外交、科技、文化、宗教、法律、基础设施、危机与战争，在数百年的时间里让一个国家从城邦成长为能长期存续的强国。

🎮 **在线试玩**：[Imperium Aeternum Pages](https://lunora-gather.github.io/Imperium-Aeternum/)

## 一、为什么做这个游戏

市面上大部分 4X / 大战略游戏的核心是「地图涂色」——扩张永远是赚的，赢了就是赢。

但我们读历史时会发现一个反直觉的事实：**罗马不是被外敌打垮的，是被自己的规模打垮的**。扩张带来财富，也带来治理压力；改革带来进步，也带来反弹；战争带来胜利，也带来财政危机与民心流失。

**Imperium Aeternum 想模拟的正是这个**：

- 扩张越大，治理越困难（over-extension 惩罚随省数上升）
- 政策越激进，反弹越强（派系满意度、合法性都会动）
- 战争越频繁，财政和民心越危险（厌战度、军费、腐败联动）
- 短期强权可以快速成功，但长期可能崩溃
- 真正的胜利是建立一个能长期运转的国家机器——**连续 200 年稳定不崩**

这不是「我要征服世界」的游戏，是「我要让这个国家活下来」的游戏。

## 二、核心玩法循环

```
查看国家状态
  → 分析财政、人口、粮食、稳定度、外交、军事
  → 制定政策（税率 / 建设 / 征兵 / 改革 / 外交）
  → 分配行动点（每回合有限）
  → 处理事件（150 个事件，多选项有后果）
  → 进入下一回合
  → 结算国家变化 → 产生新的问题与机会
```

**每一回合 = 一年**。玩家从一个小国开始，目标是让它在数百年里存续并壮大。

## 四、五种胜利 · 六种失败

**胜利**

| 类型 | 条件 |
|------|------|
| 🏆 征服 | 控制大部分省份 |
| 💰 经济 | 国库与贸易收入达到阈值 |
| 🎭 文化 | 影响力超过所有其他国家 |
| 🏛️ 永恒帝国 | **连续 200 年稳定且未崩溃**——本游戏的终极目标 |

**失败**

国库破产连续 3 年 · 全国稳定度崩落连续 3 年 · 首都被占 · 革命成功 · 国家分裂 · 合法性归零。

## 五、五大杆组 · 治国即平衡

游戏的核心张力来自五根「杆」——每根杆有主杆 + 联动 + 阻尼，阻尼随省数上升：

| 杆组 | 拉动 | 反作用 |
|------|------|--------|
| **财政** | 高税增收 | 民心降 + 派系不满 |
| **军事** | 扩军/战争 | 军费 + 厌战 + 稳定降 |
| **扩张** | 占新省 | 治能压力 + 文化冲突 + 叛乱风险 |
| **改革** | 推新政 | 派系反弹 + 短期混乱 |
| **外交** | 结盟/贸易 | 依赖风险 + 主权让渡 |

**杆之间还有耦合**：串型（A→B→C）、环型（A↔B）、对冲（A 抵消 B）。治国不是拉一根杆，是在五根杆之间找动态平衡。

## 六、技术栈

- **TypeScript** + **React 18** + **Vite 5**
- **Zustand** 状态管理
- JSON 数据驱动（150 事件 / 24 建筑 / 32 科技 / 12 政体 / 25 政策 / 12 法律）
- 本地存档 localStorage + 版本迁移
- seeded RNG（存档可复现）
- vitest 测试（50 测试覆盖公式 / 世界生成 / 回合结算 / 烟雾）

**架构**：逻辑与 UI 分离——`engine/` 各系统独立结算，`screens/` 纯展示，`store/` 统一 `processTurn(gameState)` 入口。

## 八、项目结构

```
imperium-aeternum/
  src/
    data/          # 数据表：nations / provinces / events / technologies / buildings / policies / laws / governments / characters
    engine/        # 引擎：economy / population / politics / military / diplomacy / technology / culture / events / turn / ai / chronicle / dynasty
```

## 十、设计哲学

> 一个国家最大的敌人，往往是它自己的成功。

Imperium Aeternum 不奖励「最优解」，它奖励「克制」。你能扩张，但你该不该扩张？你能加税，但你该不该加税？你能改革，但你该不该现在改？

**真正的胜利不是打赢所有战争，而是让这个国家在你不操作的时候，依然能自己运转下去。**

这就是「永恒帝国」的含义。
